# 🚔 Crime Sentinel AI - India

A professional **Full-Stack Analytics Dashboard** designed to analyze historical crime data in India (2001-2012) and provide future insights using Machine Learning.

## 🚀 Key Features
- **Predictive Analytics:** Linear Regression models forecasting crime trends up to 2030.
- **Geospatial Mapping:** Interactive intensity maps of India.
- **Safety Scoring:** A proprietary algorithm-based safety index for every State/UT.
- **Statistical Anomaly Detection:** Identifies irregular crime patterns using Z-Score analysis.
- **Executive Reporting:** Generates professional PDF reports for stakeholders.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python-based Web Framework)
- **Data Engine:** Pandas & Numpy
- **Machine Learning:** Scikit-Learn (Linear Regression)
- **Visualization:** Plotly Express & Graph Objects
- **Reporting:** FPDF Library

## 📦 Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/crime-sentinel-india.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
